Product Issue Health Dashboard

Skills, Platforms, and Area of Expertise Leveraged:

  • Data Studio/Dashboards/PLX
  • Standard SQL
  • UX Feedback Surveys
  • API Breakage
  • Seasonal Volumes

The Certifications Program has exams that tests a users ability to successfully implement G Suite products in their workplaces. These exams contain Scenario based questions, where candidates are issued a temporarily provisoned G Suite account to perform various task in Gmail, Google Meet, YouTube, Blogger, Sheets, Forms, Classroom, e.g.

Due to the various API dependencies of 1st party products on these scenario questions, I created a dashboard to instantaneously detect and report on any API errors that occur to candidates during their exams. These errors are logged in activity logs on the Google Cloud Platform which I use to create error code tables visualizing aggregate numers.

This report shows seasonality trends and volumes of issues reported to help gauge the level of effort required of that product team to maintain product infrastructure stability.

Right click image and Open in New Tab. You can see in the month of March 2020 at the peak of COVID19, we can start to see a sharp rise due to a recent Google Drive outage, and many YouTube issues. All of these data points are collated into one master joined table, with user feedback triaged and delegated to appropriate product support leads.
